# Raanjhana Events — Cinematic Royal Wedding Experience

An immersive, scroll-driven storytelling website for **Raanjhana Events** — luxury wedding planners inspired by royal Rajasthani invitations, palace architecture, and cinematic wedding films.

## Experience Scenes

1. **Royal Invitation Entrance** — Palace arch, elephants, peacock, bird cages, floating petals
2. **Invitation Opening** — Card unfolds on scroll with golden particle burst
3. **Celebration of Culture** — Ghoomar dancers, lanterns, palace courtyard
4. **Palace Pathway** — Cinematic camera travel between scenes
5. **Grand Royal Wedding** — Signature elephant animation at palace landscape
6. **Services & Gallery** — Floating holographic cards and hanging photo frames
7. **Sunset Booking** — Lantern-lit courtyard with inquiry form

## Tech Stack

- **React** + **Vite**
- **React Three Fiber** + **Three.js** — WebGL 3D scenes
- **GSAP ScrollTrigger** — Scroll storytelling
- **Framer Motion** — UI animations
- **Lenis** — Smooth scrolling

## Quick Start

```bash
npm install
npm run dev
```

Open [http://localhost:5173](http://localhost:5173)
